Set the dimensions of the extra matrix storage.

Parameters
[in]num_rowsNumber of rows in the extra matrix storage.
[in]commCommunicator used to distribute columns of the matrix.
[in]nNumber of matrix columns to distribute.
[in]idx0Starting column of the matrix.
Redistribution of wave-functions between MPI ranks

In general, the extra storage is created in the CPU pointer. The only exception is when the data distribution doesn't change (no swapping between comm_col ranks is performed) – then the extra storage will mirror the prime storage (both on CPU and GPU).

Remap data from prime to extra storage.

Prime storage is expected on the CPU (for the MPI a2a communication). If prime storage was also allocated on the device, extra storage will be copied to device as well.
