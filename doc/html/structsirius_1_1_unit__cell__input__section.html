Detailed Description

Parse unit cell input section.

The following part of the input file is parsed:

1 "unit_cell" : {
2  "lattice_vectors" : [
3  [a1_x, a1_y, a1_z],
4  [a2_x, a2_y, a2_z],
5  [a3_x, a3_y, a3_z]
6  ],
7 
8  "lattice_vectors_scale" : scale,
9 
10  "atom_types" : [label_A, label_B, ...],
11 
12  "atom_files" : {
13  label_A : file_A,
14  label_B : file_B,
15  ...
