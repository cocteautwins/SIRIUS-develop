J\+S\+O\+N Pointer. 

A J\+S\+O\+N pointer defines a string syntax for identifying a specific value within a J\+S\+O\+N document. It can be used with functions {\ttfamily at} and {\ttfamily operator\mbox{[}\mbox{]}}. Furthermore, J\+S\+O\+N pointers are the base for J\+S\+O\+N patches.
